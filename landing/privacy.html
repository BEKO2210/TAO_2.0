Datenschutzerklärung

Kurz gesagt: Diese Website setzt keine Cookies, betreibt kein Tracking, nutzt kein Analytics-Tool und enthält keine Drittanbieter-Iframes oder -Pixel. Es werden lediglich technisch unvermeidbare Verbindungsdaten durch den Hoster (GitHub Pages) verarbeitet.

1. Verantwortlicher

Verantwortlich i. S. v. Art. 4 Nr. 7 DSGVO ist der Inhaber des GitHub-Kontos BEKO2210. Kontakt: GitHub Issues oder GitHub Security Advisory am Repository. Postanschrift siehe Impressum.

2. Erhobene Daten beim reinen Aufruf der Website

Beim Aufruf einer Seite werden vom Hoster GitHub, Inc. (88 Colin P Kelly Jr St, San Francisco, CA 94107, USA) die üblichen Server-Verbindungsdaten verarbeitet:

  • IP-Adresse des anfragenden Geräts
  • Datum und Uhrzeit des Zugriffs
  • Aufgerufene Datei / URL
  • HTTP-Statuscode und übertragene Datenmenge
  • Übermittelter User-Agent (Browser, Betriebssystem)
  • Referrer (sofern vom Browser gesendet)

Diese Verarbeitung ist technisch erforderlich, um die Website auszuliefern (Rechtsgrundlage Art. 6 Abs. 1 lit. f DSGVO, berechtigtes Interesse an der Bereitstellung des Angebots). Wir selbst erhalten weder Server-Logs noch Besucherstatistiken zu dieser Website von GitHub. Die Datenverarbeitung durch GitHub richtet sich nach deren Datenschutzhinweisen: GitHub Privacy Statement. Hinweis: Es findet ein Datentransfer in die USA statt (GitHub ist US-Anbieter); GitHub ist gemäß EU – US Data Privacy Framework zertifiziert.

3. Cookies, Local Storage, Tracking

Diese Website setzt keine Cookies und beschreibt keinen Local Storage. Es findet kein Tracking, kein Profiling, keine Reichweitenmessung statt. Es werden weder Google Analytics, Plausible, Matomo, Hotjar noch vergleichbare Tools eingesetzt. Es werden keine Werbe-Pixel, Social-Media-Buttons oder eingebettete Inhalte Dritter geladen, die ohne Klick einen Datentransfer auslösen würden.

4. JavaScript

Die Seiten imprint.html und privacy.html enthalten ein kleines, lokales JavaScript-Snippet, das ausschließlich die Sprachumschaltung (Deutsch/Englisch) durchführt. Es lädt nichts nach, sendet keine Daten und nutzt keinen Local Storage.

5. Externe Links und Schriftarten

Diese Website verlinkt auf externe Seiten (z. B. das GitHub-Repository, Bittensor-Dokumentation). Beim Klick auf einen Link werden Daten an den jeweiligen Anbieter übertragen, dessen Datenschutzerklärung dann maßgeblich ist.

6. Daten beim Nutzen der TAO-Swarm-Software (lokal auf deinem Rechner)

Die Software selbst läuft vollständig lokal auf deinem Rechner. Sie enthält keine Telemetrie, kein Crash-Reporting und keine Update-Pings. Beim Aufruf mit --live stellt sie ausgehende HTTPS-Verbindungen zu den folgenden öffentlichen Endpoints her, ausschließlich um öffentliche Daten zu lesen:

Diese Verbindungen werden nur hergestellt, wenn du die Software explizit im Live-Modus startest. Die jeweiligen Anbieter haben eigene Datenschutzbestimmungen, an die du dich beim Live-Modus faktisch hältst.

Privacy Policy

In short: this website sets no cookies, runs no tracking, uses no analytics tool, and embeds no third-party iframes or pixels. Only technically unavoidable connection data is processed by the host (GitHub Pages).

1. Data controller

Controller within the meaning of Art. 4 (7) GDPR is the holder of the GitHub account BEKO2210. Contact: GitHub issues or GitHub Security Advisory on the repository. Postal address: see imprint.

2. Data processed when you simply visit the site

When you load a page, the host GitHub, Inc. (88 Colin P Kelly Jr St, San Francisco, CA 94107, USA) processes the usual server connection data:

  • IP address of the requesting device
  • Date and time of access
  • URL / file requested
  • HTTP status code and bytes transferred
  • User-Agent string (browser, OS)
  • Referrer (if your browser sends one)

This processing is technically necessary in order to deliver the site (legal basis: Art. 6 (1) lit. f GDPR — legitimate interest in providing the service). We ourselves do not receive server logs or visitor analytics for this site from GitHub. GitHub's processing is governed by its own privacy notices: GitHub Privacy Statement. Note: data is transferred to the USA (GitHub is a US provider); GitHub is certified under the EU–US Data Privacy Framework.

3. Cookies, local storage, tracking

This site sets no cookies and writes no local storage. There is no tracking, no profiling, no audience measurement. We use neither Google Analytics nor Plausible, Matomo, Hotjar or any comparable tools. We do not load advertising pixels, social-media buttons, or embedded third-party content that would trigger a data transfer without a click.

4. JavaScript

The pages imprint.html and privacy.html contain a small local JavaScript snippet that only switches the language (German/English). It does not load anything, sends no data, and uses no local storage.

5. External links and fonts

The site links to external pages (e.g. the GitHub repository, Bittensor documentation). When you click a link, data is transferred to the respective provider, whose privacy policy then applies.

No external fonts are loaded; the site uses only system fonts on your device.

6. Data when using the TAO Swarm software (locally on your machine)

The software itself runs fully locally on your machine. It contains no telemetry, no crash reporting, and no update pings. When invoked with --live, it makes outbound HTTPS connections to the following public endpoints, solely to read public data:

  • entrypoint-finney.opentensor.ai — Bittensor mainnet node

  • api.coingecko.com — TAO price and 24h data

These connections are only established when you explicitly start the software in live mode. Each provider has its own privacy terms which you are de facto subject to when using live mode.

  • Access (Art. 15 GDPR)

  • Restriction of processing (Art. 18 GDPR)
  • Data portability (Art. 20 GDPR)
  • Objection to processing (Art. 21 GDPR)
  • Complaint to a supervisory authority (Art. 77 GDPR)

Note: because we ourselves store no logs or other identifying data for this site, an access request can only confirm that no data is held by us. For the host (GitHub), please contact GitHub directly.
