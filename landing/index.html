What it is

A read-only research swarm.
Not a trading bot.

01

15 specialized agents

System check, protocol research, subnet discovery and scoring, watch-only wallet, market analysis, risk review, miner / validator engineering, training, infra, dashboard, fullstack, QA, docs.

02

Gate before route

Every action is classified SAFE / CAUTION / DANGER before the router ever sees it. DANGER outputs a plan only — execute_trade, sign_transaction, stake, unstake, swap_coldkey, reveal_seed are all blocked by construction.

03

Plug-in architecture

Drop your own *_agent.py in a folder, set TAO_PLUGIN_PATHS, done. Or ship as a pip package via the tao.agents entry point. Plug-ins go through the same gate as built-ins. No privilege escalation.

04

Wallet stays in your hands

Default mode is NO_WALLET. WATCH_ONLY accepts public SS58 addresses only. MANUAL_SIGNING emits unsigned plans — you sign on a device we never touch. The system never asks for, stores, or transmits seed phrases or private keys.

05

Live data, real fallbacks

Bittensor SDK 10.x against mainnet finney for 129+ subnets with rich identity. Subscan for wallet snapshots. CoinGecko for prices. Each path falls back to mock fixtures with a labelled _meta.fallback_reason when the upstream is unreachable — never silent.

06

Local-only, no telemetry

Everything runs on your machine. SQLite for state. No analytics. No vendor SDKs that phone home. Outbound HTTPS only to the data endpoints you explicitly opt into via --live.

How it runs

Three commands.

1

Install

git clone https://github.com/BEKO2210/TAO_2.0.git
cd TAO_2.0
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt -r requirements-dev.txt
pip install -e .
2

Run mock

tao-swarm subnets
tao-swarm score 1
tao-swarm risk --content "Schedule_coldkey_swap to 5XYZ..."
# → DANGER: coldkey_swap_social_engineering CRITICAL
3

Go live

pip install bittensor
tao-swarm --live subnets   # 129 real Mainnet subnets
tao-swarm --live market    # real CoinGecko price + 24h
tao-swarm --live run --task '{"type": "subnet_discovery"}'
