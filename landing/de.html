Was es ist

Read-only Recherche-Schwarm by default.
Auto-Trader by deliberate choice.

01

15 spezialisierte Agenten

System-Check, Protokoll-Recherche, Subnet-Discovery und -Scoring, Watch-Only-Wallet, Marktanalyse, Risk-Review, Miner-/Validator-Engineering, Training, Infra, Dashboard, Fullstack, QA, Doku.

02

Gate vor Routing

Jede Aktion wird als SAFE / CAUTION / DANGER klassifiziert, bevor der Router sie sieht. In Read-only-Modi liefert DANGER nur einen Plan. Im opt-in AUTO_TRADING-Modus geht DANGER an die auditierte Execution-Pipeline nur wenn Kill-Switch aus, Position-Cap gesetzt und Daily-Loss-Limit gesetzt sind.

03

Plug-in-Architektur

Lege deinen eigenen *_agent.py in einen Ordner, setze TAO_PLUGIN_PATHS — fertig. Oder shippe als pip-Package via tao.agents-Entry-Point. Plug-ins durchlaufen das gleiche Gate wie Built-ins. Kein Privilege-Escalation.

04

Vier Wallet-Modi, opt-in

Default ist NO_WALLET. WATCH_ONLY akzeptiert nur öffentliche SS58-Adressen. MANUAL_SIGNING liefert ungezeichnete Pläne, du signierst extern. AUTO_TRADING nutzt einen einzelnen Hot-Key mit hartem Cap, Kill-Switch und Daily-Loss-Stop. Seed-Phrases und Coldkey-Private-Keys werden nie angefragt, gespeichert oder übertragen.

05

Live-Daten mit echten Fallbacks

Bittensor SDK 10.x gegen Mainnet finney für 129+ Subnets mit Identity. Subscan für Wallet-Snapshots. CoinGecko für Preise. Jeder Pfad fällt auf Mock-Fixtures zurück mit einem gelabelten _meta.fallback_reason, wenn der Upstream nicht erreichbar ist — niemals stillschweigend.

06

Lokal-only, keine Telemetrie

Alles läuft auf deinem Rechner. SQLite für State. Keine Analytics. Keine Vendor-SDKs, die nach Hause telefonieren. Outbound-HTTPS nur zu den Daten-Endpoints, die du via --live explizit opt-in aktivierst.

Wie es läuft

Drei Befehle.

1

Installieren

git clone https://github.com/BEKO2210/TAO_2.0.git
cd TAO_2.0
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt -r requirements-dev.txt
pip install -e .
2

Mock laufen lassen

tao-swarm subnets
tao-swarm score 1
tao-swarm risk --content "Schedule_coldkey_swap to 5XYZ..."
# → DANGER: coldkey_swap_social_engineering CRITICAL
3

Live gehen

pip install bittensor
tao-swarm --live subnets   # 129 echte Mainnet-Subnets
tao-swarm --live market    # echter CoinGecko-Preis + 24h
tao-swarm --live run --task '{"type": "subnet_discovery"}'

Wir haben keinen Token und werden keinen launchen. Wir verwalten keine fremden Gelder. Auto-Trading ist ein Single-User-Feature auf deinem Rechner mit deinem eigenen Hot-Key.
